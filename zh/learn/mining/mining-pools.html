矿池 ​

Qubic-Li ​

简介：

在 Qubic 网络中，Qubic-Li 池是当前规模最大的矿池，它以稳定性和同时支持 CPU 与 GPU 开采的功能而著称。该池能够兼容 LINUX、WIN 和 HIVEOS 等操作系统环境。

网站：https://web.qubic.li/

矿池网站：https://app.qubic.li/main/mining/pool

Github：https://github.com/qubic-li

Discord：https://discord.com/invite/2vDMR8m

Quick Start：https://github.com/qubic-li/client/blob/main/README.md

Apool ​

简介：

Apool 是网络中第二大的挖矿池，运作在 PPLNS 方式下。每个阶段结束后的每周都会发放数字货币。它兼容多种操作平台，并以高效的 GPU 能效比例著称。当前的池子费用设定为 10%。

网站：https://www.apool.io/

Github：https://github.com/apool-io

Discord：https://discord.com/invite/jpV4V2dUG6

Quick Start：https://apool.gitbook.io/help

Minerlab ​

简介：

Minerlab-io 提供最快速的代币分配，Minerlab-io 池热忱于与高哈希计算能力的用户交流，探讨与池主之间的费用协商。它采用的是与 Qubic-Li 相同的挖矿软件。对于想要从 Qubic-Li 转移至 Minerlab 池的用户来说，只需按照矿池提供的一系列操作指南即可轻松更新 Token 和矿工名称，实现快速过渡。

网站：https://qubic.minerlab.io/

Discord: https://discord.gg/CtT3mtxSpY

Quick Start: https://qubic.minerlab.io/profile

Solutions ​

简介：

Solution Pool 在每个纪元结束一周后分发代币。目前，该池提供混合型挖掘工具，利用 GPU 加速来提升 CPU 性能，从而实现更高的能源效率比。

网站：http://qubic.solutions/

Github: https://github.com/Qubic-Solutions

Discord: https://discord.com/invite/zTrdShyQu2

Quick Start: https://github.com/Qubic-Solutions/.github/blob/main/profile/README.md

Qubicmine-Pro ​

简介：

Qubicmine-Pro 矿池基于 PPLNS 运作，在每个纪元结束后一周分配硬币。它使用与 Qubic-Li 相同的挖矿软件，并且需要一定程度的网络稳定性。

网站：https://qubicmine.pro/

Github: https://github.com/hackerby888

Discord: https://discord.com/invite/YVPzYhQ66N

Quick Start: https://qubicmine.pro/get-started
